# MAD201-ASSIGNMENT04-Jennyfer
simple React Native mobile portfolio application built with Expo and Expo Router. The app showcases a developer's personal profile, projects, and contact information through a clean and modern interface.

Features Overview

Home Screen (index.tsx)
Displays the user’s profile image, name, and short bio
Includes buttons to navigate to the Projects and Contact screens
Acts as the entry point of the app

Projects Screen (projects.tsx)
Shows a scrollable list of personal projects
Uses the reusable ProjectCard component for consistent layout
Each card displays:
Project title
Short description
A “Learn More” placeholder button

Contact Screen (contact.tsx)
Provides simple contact options
Contains links/buttons for:
LinkedIn
GitHub
Email
Allows users to reach the developer quickly and easily

Modal Screen (modal.tsx)
Demonstrates modal navigation using Expo Router
Contains a title and a link to return back to the Home screen
Uses themed UI components for styling consistency

ProjectCard Component
A reusable UI card component
Displays project title, description, and a button
Used within the Projects screen to show multiple projects in a consistent style

Tech Stack
React Native
Expo
TypeScript
Expo Router
Custom UI Components
